# Quick description

It is a nvim config, that use gruvbox theme, which is setup for:

- markdown editing
- web (html,css,js,php,liquid)
- python
- c (to do: setup the debugger)
- c#

It use lazy, mason, conform ...
It use yazi for file managament and as a file explorer
It support code folding
And have some cool features
